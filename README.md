# Show and Tell

**Inspired by: The Yard Sale Model - Exploring the Mathematical Inevitability of Wealth Inequality**

Welcome to **Show and Tell**, an interactive platform that visualizes the behaviors of complex systems through simulations. This project transforms theoretical models into engaging, intuitive, and highly interactive experiences.

This work was inspired by the research described in the following article: [The Mathematics of Poverty, Inequality, and Oligarchy](https://www.siam.org/publications/siam-news/articles/the-mathematics-of-poverty-inequality-and-oligarchy).

---

## Vision

Our guiding philosophy:  
**"Intuition through visualization, rigor through interaction."**

## Motivations

Inspired by academic research, such as the "Yard Sale Model" (exploring how perfectly fair trades can lead to income inequality), **Show and Tell** aims to make abstract mathematical truths tangible.

## Features

- **Interactive Simulations**  
  Engage with live examples like the Yard Sale Model to witness concepts like oligarchy formation and wealth inequality.
  
- **Playground + Analysis**  
  Start with experimentation via interactive settings, and then dive deep into the data-driven tabs like "Analysis" or "Theory." Understand emergent factors behind inequality—such as "absorbing barriers."

- **Progressive Engagement**  
  From visualization to detailed theory: Choose your own level of depth, whether just watching simulations or rigorously analyzing data.

## Getting Started

Use the following commands to run/experiment with the platform locally:

```bash
npm install     # Install dependencies
npm run dev     # Start the development server
npm run build   # Build the project for production
npm test        # Run the tests (via Playwright)
```
